This creates an animation of two units attacking each other. In javafx and with heavy use of `javafx.animation.Animation`s.

The entry point is {@link name.rayrobdod.fightStage.BattleAnimation#buildAnimation}

The primary extension point is implementing {@link name.rayrobdod.fightStage.UnitAnimationGroup} and {@link name.rayrobdod.fightStage.SpellAnimationGroup}.
